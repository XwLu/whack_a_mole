# Arduino打地鼠游戏

基于Arduino UNO开发的经典打地鼠游戏。

## 功能特点

- 🎮 经典打地鼠游戏体验
- 🔢 两位数码管显示实时得分
- ⏱️ 60秒游戏时长
- 🔄 自动刷新地鼠位置（超时或被击中后）
- 💪 难度递增（随着分数增加，地鼠显示时间缩短）
- 🔁 游戏过程中，同时按下3个地鼠按钮回到准备状态
- 🔁 准备状态下，同时按下任意2个按钮开始游戏
- 🔁 游戏正常结束后，显示最终得分并回到准备状态

## 硬件需求

### 核心组件
- Arduino UNO R3 开发板 × 1
- 四位数码管（IIC驱动） × 1
- 带LED的自动复位按钮 × 5
- 继电器 × 5
- 12v电源 × 1
- 12v降5v降压模块 × 1

## 硬件连接图

### IIC数码管连接

| 数码管引脚 | Arduino引脚 | 功能说明 |
|------------|-------------|----------|
| VCC        | 5V          | 电源正极 |
| GND        | GND         | 电源负极 |
| SDA        | A4          | IIC数据引脚 |
| SCL        | A5          | IIC时钟引脚 |

**注意：** IIC数码管通常使用标准的IIC接口，只需4根线连接即可。

### 地鼠按钮和LED连接

#### LED控制引脚（输出）
| 组件 | Arduino引脚 | 说明 |
|------|-------------|------|
| 地鼠1 LED | 12 | LED正负极直接接12v点亮 |
| 地鼠2 LED | A0 | LED正负极直接接12v点亮 |
| 地鼠3 LED | A1 | LED正负极直接接12v点亮 |
| 地鼠4 LED | A2 | LED正负极直接接12v点亮 |
| 地鼠5 LED | 2 | LED正负极直接接12v点亮 |

#### 按钮引脚（输入）
| 组件 | Arduino引脚 | 说明 |
|------|-------------|------|
| 地鼠1按钮 | 13 | 上拉输入 |
| 地鼠2按钮 | A3 | 上拉输入 |
| 地鼠3按钮 | 4 | 上拉输入 |
| 地鼠4按钮 | 5 | 上拉输入 |
| 地鼠5按钮 | 3 | 上拉输入 |

**复位功能**：同时按下任意3个或更多地鼠按钮即可回到准备状态

**引脚说明**：所有按钮引脚均避免了与IIC数码管的SDA(A4)和SCL(A5)引脚冲突，充分利用了原来数码管占用的引脚资源。

**连接说明：**
- LED：正极连接继电器COM端，负极接地
- 继电器：COM端接LED，NO端接12v电源，VCC接5v电源，GND接地，控制端接Arduino引脚
- 按钮：一端连接Arduino按钮引脚，另一端接地（代码已启用内部上拉电阻）
- 12v电源降5v后驱动Arduino和继电器

## 软件实现

### 主要功能模块

1. **数码管驱动**：实现4位数码管的动态显示
2. **按钮控制**：检测地鼠按钮和复位按钮的状态
3. **游戏逻辑**：
   - 上电或复位后进入准备状态，同时按下2任意个按键，数码管开启倒计时
   - 倒计时结束，随机显示地鼠（LED亮起）
   - 检测玩家是否击中地鼠
   - 计分系统
   - 60秒倒计时
   - 游戏结束判定：60秒后，游戏结束，显示最终得分，并回到准备状态
   - 游戏过程中，同时按下3个地鼠按钮，游戏立即结束，回到准备状态

## 注意事项

1. **LED限流电阻**：必须为所有LED串联220Ω电阻，否则可能损坏LED或Arduino，我用的是淘宝上买的内置了电阻的LED，可以直接接12v点亮
2. **按钮数量**：可以根据实际硬件调整`MAX_MOLES`常量
4. **随机数生成**：使用模拟引脚A5作为随机源，确保该引脚未连接任何传感器
5. **消抖处理**：谨慎增加delay函数，可能导致程序空等，短时间按下按钮无法被检测到（表现为明明打中了地鼠但是没有得分）
